# PMMKickstarter AI Field Guide

This repository hosts the content layer for The Agentic PMM Field Guide by [PMMKickstarter.com](https://pmmkickstarter.com).

The field guide is published at [pmmkickstarter.com/the-agentic-pmm/](https://pmmkickstarter.com/the-agentic-pmm/). This repo makes the content available in a format that AI tools can query directly.

## What's Here

### `/field-guide`

30 sections of The Agentic PMM Field Guide in clean Markdown, plus a `manifest.json` that indexes all sections with titles, section numbers, file URLs, and topic tags.

The manifest is the entry point for AI tools — fetch it first to understand what's available, then fetch only the sections relevant to the query.

**Manifest URL:**
```
https://raw.githubusercontent.com/admannz/pmmkickstart/main/field-guide/manifest.json
```

## The Cowork Skill

A `field-guide.skill` file is available for download from [pmmkickstarter.com](https://pmmkickstarter.com). Install it in Cowork to query the field guide directly in conversation — free-form questions, topic lookups, or guided reading paths based on your situation.

## About the Field Guide

The Agentic PMM is a practical field guide for founders and product marketers navigating the PMM discipline in the Age of AI. It covers:

- **Section 1 — The Discipline:** What PMM is, where it sits, how to hire for it
- **Section 2 — The Craft:** Messaging, ICP, research, competitive intelligence, sales enablement, launch management
- **Section 3 — PMM in the Age of AI:** What's changed, what hasn't, and how to use AI well
- **Section 4 — Building Your Agentic PMM Function:** Checklists, AI briefs, metrics, scaling
- **Section 5 — The PMM Career:** Currency, pivots, first 90 days, leading a team

Written by Adam at [PMMKickstarter.com](https://pmmkickstarter.com).
